# Wine Quality Prediction

This project predicts wine quality using machine learning models based on chemical properties from the UCI Wine Quality dataset.

## Models Used
- KNN
- Decision Tree
- Random Forest
- XGBoost

## Result
Random Forest achieved the best overall performance, with accuracy of about 87% on red wine data.

## Files
- `wine_quality_analysis.ipynb` : notebook for preprocessing, training, and evaluation

## Dataset

This project uses the Wine Quality dataset from the UCI Machine Learning Repository.

Files included:
- winequality-red.csv
- winequality-white.csv

Source:
https://archive.ics.uci.edu/ml/datasets/Wine+Quality
